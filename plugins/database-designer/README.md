# Database Designer / 数据库设计器

[English](#english) | [中文](#中文)

## English

### Overview

Comprehensive database design tool that creates complete database schemas with tables, fields, indexes, and ER diagrams. Generates executable SQL scripts and DrawDB-compatible files for visual design.

### Features

- **Multiple Database Support**: MySQL (default), PostgreSQL, SQL Server
- **Complete Documentation**: Markdown design docs with detailed comments
- **Visual ER Diagrams**: DrawDB JSON and DBML format export
- **Best Practices**: Logical relationships, realistic field sizes, optimized indexes
- **Production-Ready**: All tables and fields include Chinese comments

### Installation

```bash
claude plugin install github:shining319/claude-code-single-person-workflow/plugins/database-designer
```

### Usage

Use this plugin when you need to:
- Design database schemas from scratch
- Create data models for business domains
- Generate SQL scripts for implementation
- Produce ER diagrams for documentation

### Output Formats

1. **Markdown Design Document**: Complete schema documentation
2. **SQL Script**: Executable database creation script
3. **DrawDB JSON**: Import into DrawDB for visual editing
4. **DrawDB DBML**: Alternative DrawDB format
