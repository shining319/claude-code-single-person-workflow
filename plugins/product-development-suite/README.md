# Product Development Suite / 产品开发套件

[English](#english) | [中文](#中文)

## English

### Overview

Complete all-in-one suite for single-person product development. Includes all five specialized skills in one convenient package: academic writing, database design, product management, UI/UX design, and solution architecture.

### Included Skills

1. **Academic Writing Style** - Professional documentation and reports
2. **Database Designer** - Complete database schema design with ER diagrams
3. **Product Manager** - Requirements analysis and PRD creation
4. **UI Designer** - Interface design and specifications
5. **Solution Architect** - Technical architecture and deployment planning

### Features

- **Complete Toolkit**: Everything you need for product development
- **Integrated Workflow**: All skills work together seamlessly
- **Single Installation**: Get all capabilities at once
- **Consistent Quality**: Same high-quality output across all skills

### Installation

```bash
claude plugin install github:shining319/claude-code-single-person-workflow/plugins/product-development-suite
```

### Use Cases

This suite is perfect for:
- Solo entrepreneurs building complete products
- Technical founders designing and documenting systems
- Product designers creating comprehensive specifications
- Developers who need end-to-end product capabilities

### What You Can Do

With this suite, you can:
- ✅ Define product requirements and create PRDs
- ✅ Design technical architecture and choose tech stacks
- ✅ Create complete database schemas with ER diagrams
- ✅ Design UI interfaces with detailed specifications
- ✅ Write professional technical documentation
