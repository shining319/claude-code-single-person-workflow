
## 示例：老年用户画像

### 用户画像：银发族用户

**基本信息**
- 年龄段：60-75岁
- 职业/身份：退休人员
- 技术熟练度：低
- 使用场景：日常生活、与家人沟通

**目标与动机**
- 主要目标：完成基本任务（如缴费、购物）
- 次要目标：与子女保持联系
- 深层动机：保持独立、不给子女添麻烦

**痛点与挑战**
- 当前解决方案的不足：字体小、操作复杂、专业术语多
- 核心痛点：害怕操作失误、记不住流程
- 情感诉求：需要被尊重、获得耐心指导

**行为特征**
- 使用频率：低频但有刚需
- 关键触点：子女推荐、社区宣传
- 决策因素：简单易用 > 功能丰富
