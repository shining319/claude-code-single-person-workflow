## 用户故事模板

```
作为 [用户角色]，
我想要 [完成某个任务/获得某个功能]，
以便 [实现某个目标/获得某个价值]。

验收标准：
- Given [前置条件]
- When [用户行为]
- Then [期望结果]
```

## 功能优先级说明

**P0（Must Have）**：不做产品无法上线，核心价值体现

**P1（Should Have）**：重要但可在后续版本补充

**P2（Could Have）**：锦上添花，资源充足时考虑

**P3（Won't Have）**：本版本明确不做
