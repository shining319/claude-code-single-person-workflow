# 常见页面类型参考

## 网页端页面

### 首页/Landing Page
- 头部导航栏
- Hero区域（大图+标题+CTA按钮）
- 功能/特性展示区
- 客户案例/社会证明
- 定价区域（可选）
- 页脚

### 登录/注册页
- Logo
- 表单区域（用户名、密码、验证码等）
- 第三方登录按钮
- 忘记密码/注册切换链接
- 服务协议复选框

### 仪表盘/Dashboard
- 顶部导航栏
- 侧边导航菜单
- 数据卡片区域
- 图表展示区
- 快捷操作区
- 最近动态/消息

### 列表页
- 页面标题+操作按钮
- 搜索+筛选栏
- 数据表格/卡片列表
- 批量操作栏
- 分页器

### 详情页
- 面包屑导航
- 主信息区（标题、状态、基本信息）
- Tab切换内容区
- 操作按钮区
- 相关内容推荐

### 表单页
- 页面标题+说明
- 分步骤/分组表单
- 字段验证提示
- 提交/取消按钮
- 草稿保存功能

### 设置页
- 侧边设置菜单
- 分组设置项
- 开关/选择器控件
- 保存/重置按钮

## 移动端页面

### 首页/主页
- 状态栏
- 搜索栏/标题栏
- Banner/轮播图
- 功能入口网格
- 内容列表/Feed流
- 底部导航栏

### 列表页
- 导航栏（返回+标题+操作）
- 搜索/筛选
- 列表内容（卡片/行）
- 下拉刷新+上拉加载
- 悬浮操作按钮

### 详情页
- 导航栏（返回+分享+更多）
- 主图/视频区
- 信息区域
- 评论/互动区
- 底部操作栏

### 个人中心
- 头像+用户信息卡片
- 数据统计区
- 功能菜单列表
- 设置/退出入口

### 购物车/订单页
- 导航栏
- 商品列表
- 价格汇总区
- 优惠券/备注
- 底部结算栏

### 消息/通知页
- 导航栏+清空操作
- 消息分类Tab
- 消息列表
- 空状态提示

## 通用组件页面

### 弹窗/Modal
- 遮罩层
- 弹窗容器（标题+内容+操作）
- 关闭按钮
- 操作按钮组

### 抽屉/Drawer
- 遮罩层
- 侧滑面板
- 标题栏
- 内容区域
- 操作按钮

### Toast/Snackbar
- 图标
- 提示文字
- 操作按钮（可选）
- 自动消失动效

### 空状态
- 插图
- 提示文字
- 操作按钮

### 加载状态
- 骨架屏
- 加载动画
- 进度指示

### 错误页面
- 错误插图/图标
- 错误码+说明
- 返回/重试按钮
