## 字体规范

### 中文字体推荐
- 苹方 (PingFang SC) - Apple设备
- 思源黑体 (Source Han Sans) - 开源通用
- 微软雅黑 - Windows设备
- 阿里巴巴普惠体 - 免费商用

### 字号层级
```
H1 标题：28-36px, Bold
H2 副标题：22-28px, Semibold
H3 小标题：18-22px, Medium
正文：14-16px, Regular
辅助文字：12-14px, Regular
```

### 行高建议
- 标题：1.2-1.4
- 正文：1.5-1.75
- 辅助文字：1.4-1.6

## 交互状态

### 按钮状态
- Default：正常显示
- Hover：亮度+10%或添加阴影
- Active/Pressed：亮度-10%
- Focus：添加焦点环(2px outline)
- Disabled：透明度50%，禁止点击

### 输入框状态
- Default：灰色边框
- Focus：主色边框+阴影
- Error：红色边框+错误提示
- Disabled：灰色背景，禁止输入
- Readonly：略深背景，允许选中

## 动效建议

### 过渡时长
```
快速：150ms (悬停、按压)
正常：250ms (展开、收起)
慢速：350ms (页面切换)
```

### 缓动函数
```
ease-out：元素进入
ease-in：元素离开
ease-in-out：状态切换
```

### 常用动效
- 按钮悬停：scale(1.02) + box-shadow
- 卡片悬停：translateY(-2px) + box-shadow
- 下拉展开：height + opacity
- 页面切换：fade + slide
